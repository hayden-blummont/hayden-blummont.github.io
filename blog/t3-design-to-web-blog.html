What is a responsive site and why is it important?

A responsive site is one that adjusts based on the resolution of a screen to render the web page well on an array of different screens with varying sizes. This means that the majority of users will have more consistency across devices and there doesn't necessarily have to be a totally different mobile site.

What is a mobile first design and why is it important?

Mobile first design is when you design what the mobile web page will look like first and then build up to bigger screens, adjusting and adding features as the resolution increases. This method means that you don't need a separate mobile site which saves time (and money). It's also a good way to get you to prioritise what's really important and how you'll show that first.

What is a framework and what are some of their pros and cons?

Frameworks in this context are kind of like templates for web pages. Some of them are simple and have just very basic code while others are more complete.

What is a wireframing and why do we use it?

A wireframe is a really simplified visual representation of the website that shows the basic elements and where they will be on the web page. This allows for clear planning and easy prioritisation.
