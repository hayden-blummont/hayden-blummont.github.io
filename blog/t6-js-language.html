Technical Blog Part 6
In the JavaScript language, describe all the use cases for:

parentheses () - These are used mostly group or contain expressions and parameters. When grouping expressions, it tells the computer when to execute each part like in x = (6 + 3) * 4;
If the function has no parameters, the parentheses are left blank eg. function example()

brackets [] - Brackets are used mostly to wrap arrays (lists) or objects from within arrays that are used in other statements.

braces {} - These are used to wrap statements that make up a function. They define the start and end of a function. They can also be nested and separate blocks of code.

single quotes '' AAAAAAND double quotes "" - These are probably the trickiest to explain but for most purposes, these two are interchangeable and used to wrap things like strings and classes.
Double quotes are best for things that might need to contain a single quote inside or an apostrophe. It can be useful to use single quotes when you're going to need a double quote inside but there are other ways around this.
